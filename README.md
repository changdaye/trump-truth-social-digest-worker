# trump-truth-social-digest-worker

A Cloudflare Worker project for monitoring Donald Trump's Truth Social posts,
producing a Chinese digest every two hours, pushing brief summaries to Feishu,
and archiving detailed Markdown reports to object storage.

## Status

Design approved. Implementation has not started yet.
